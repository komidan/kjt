### Commands
`/createjump <name>` - Creates a jump.\
`/deletejump <name>` - Deletes a jump.\
`/setjump <name>` - Sets jump, teleport player to location. Don't pass <name> for jump list.\
`/jumps` - Opens the jumps list.
